
# YOLOv8 Human Detection & Counting (Windows 11)

## Features
- YOLOv8 (Ultralytics) for fast & accurate human detection
- Supports Webcam, Video File, IP Camera, RTSP Stream
- Simple Tkinter GUI
- Windows 11 compatible

## Setup (Windows 11)
```bash
python -m venv venv
venv\Scripts\activate
pip install --upgrade pip
pip install ultralytics opencv-python pillow
```

## Run
```bash
python app.py
```

## Notes
- Internet required only once to download YOLOv8 weights
- Press Q to stop video
